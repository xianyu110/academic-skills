# Academic Skills 学术 Skills 仓库

这是一个把微信公众号「学术AI大模型」中与 `Skill` / `Skills` 相关内容整理成的学习型资料仓库，并附带 GitHub Pages 静态索引页。

## 内容

- `articles/`：已下载并转存的 22 篇 Skills 相关文章 Markdown。
- `skills/INDEX.md`：按科研流程分类的 Skills 文章索引。
- `data/articles.json`：结构化文章元数据。
- `docs/`：GitHub Pages 静态站点文件。

## 分类

- 文献阅读与知识组织：2 篇
- 选题与论文构思：4 篇
- 论文写作与语言打磨：7 篇
- 审稿与反馈：2 篇
- 汇报与展示：1 篇
- Skills 方法论：3 篇
- 其他：1 篇
- 复现与科研工程：2 篇

## 本地预览

```bash
python3 -m http.server 8000 -d docs
```

然后打开 http://localhost:8000 。
